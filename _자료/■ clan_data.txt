hascastle
성 소유여부 castle 테이블의 castle_id를 입력한다.
1 = 켄트성
2 = 오크성
3 = 윈다우드성
4 = 기란성
5= 하이네성
6 = 아덴성
7 = 디아드 요새

hashouse
아지트 소유여부 house 테이블의 house_id를 입력한다.
262145	= 기란아지트 1

alliance
동맹인 혈맹의 clan_id를 입력한다.

bot
true = 군주 앞에서 /가입 활성화
false = 군주 앞에서 /가입 비활성화

join_setting
0 = 혈맹원을 모집하는 상태
1 = 혈맹원을 모집하지 않는 상태

join_type
0 = 즉시 가입
1 = 승인 가입
2 = 암호 가입
3 = 초대 가입

total_m
혈맹의 전체 혈원(멤버)가 몇명인지 나타낸다.

current_m
혈맹의 접속한 혈원(멤버)이 몇명인지 나타낸다.
